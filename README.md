## Dependencies
 jQuery.
## Markup:
So far, it only supports images inside the main container. Of course I plan on changing that later.
  ```html
  <div id="xexeuContainer">
    <img src="images/example1.png">
    <img src="images/example1.png">
    <img src="images/example1.png">
  </div>
  ```
## Initialization:
```javascript
$('#xexeuContainer').xexeuCarousel();
```
